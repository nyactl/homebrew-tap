# homebrew-tap

Homebrew tap for [nyactl](https://github.com/nyactl) tools.

## Usage

```sh
brew install nyactl/tap/<formula>
```

The tap is added automatically — no separate `brew tap` step needed.

## Formulae

| Formula | Description |
|---------|-------------|
| [todoist-cli](https://github.com/nyactl/todoist-cli) | Fast, minimal Todoist CLI with a local SQLite cache |

## Manual tap

If you want to tap first and install separately:

```sh
brew tap nyactl/tap
brew install todoist-cli
```
